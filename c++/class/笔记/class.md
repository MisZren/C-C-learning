```
//c++中，对结构体进行了扩展，允许在结构体中写函数
//访问权限：对结构体/类中的数据或函数的访问加以限制
//1. public: 公有权限，可以随意访问（类里，类外）
//2. protected: 保护权限， 与继承有关，（限制儿子可以访问）
//3. private: 私有权限，自己（类里）可以访问

// 结构体与类的区别： 默认的访问权限不同
// 1. 结构体：默认公有 public
// 2. 类 ： 默认私有 private
// 通常数据私有，部分成员函数公有（接口函数）
// 对于访问权限的检查是编译期而不是运行期

// 类的大小：只计算成员变量占的空间
```
